Advanced Settings

Overview

The Advanced menu provides access to the following pages and settings:
  • Networks - mobile network settings
  • Manual DNS - specify the DNS to use
  • SIM - SIM PIN operations
  • Firewall - VPN passthrough, IPv6 transparent mode, DMZ
  • MAC Filter - determine which devices are able to connect to the Jetpack's Wi-Fi network
  • LAN - IPv4, IPv6 settings
  • Port Filtering - Define which outgoing Internet traffic is allowed
  • Port Forwarding - Allow specified incoming Internet traffic to reach a specified connected device.

Note that for normal operation, and normal situations, none of these settings need to be changed from their default values. Only change these settings if your particular situation requires changes.

Screen Sharing

Overview

This feature allows you to share your computer screen will other connected devices. To share your computer screen you must have Java installed if using Chrome, Firefox or similar browsers. Java is not required if using Internet Explorer. Viewers can use any Web browser on any device.

This screen will look different depending on whether or not you have signed in to the Jetpack Admin website (this site) using the Jetpack Admin Password. If signed in, the ON/OFF control, and the Main Menu, will be visible. Otherwise, there elements are hidden.

ON/OFF Control

This control is only visible if you have signed in to the Jetpack Admin website (this site) using the Jetpack Admin Password.

Use this setting to turn off the Screen Sharing feature, if desired.

Start or Join a Screen Sharing Session

Current Session

This indicates if somebody is currently sharing their screen. If nobody is sharing, you can start a screen-sharing session by entering your name and clicking the "Start Sharing" button.

If a screen-sharing session is in progress, you can start viewing the Host's screen by clicking the "Start Viewing" button.

If are currently a Viewer, and now wish to share your screen (become the Host), you must wait until the current host has ended their screen-sharing session.

Show my name as

If you are sharing your screen (you are the Host), this name is shown to the Viewers. If you are viewing the Host's screen, this name is shown to the Host.
